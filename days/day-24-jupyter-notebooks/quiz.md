# Day 24 Quiz: Jupyter Notebooks

## Questions

### 1. What are the two main types of cells in Jupyter?

**Your answer:**


**Correct answer:** Code cells (for Python code) and Markdown cells (for documentation). Code cells execute Python, Markdown cells render formatted text, headings, lists, etc.

---

### 2. How do you run a cell in Jupyter?

**Your answer:**


**Correct answer:** Press Shift+Enter to run the current cell and move to the next one, or Ctrl+Enter to run without moving. You can also use the Run button in the toolbar.

---

### 3. What does `%matplotlib inline` do?

**Your answer:**


**Correct answer:** It's a magic command that displays matplotlib plots directly in the notebook below the code cell, rather than in a separate window.

---

### 4. Why use Jupyter for data analysis?

**Your answer:**


**Correct answer:** Jupyter allows interactive exploration, combines code with documentation, displays visualizations inline, enables easy sharing, and lets you iterate quickly on analysis.

---

### 5. How do you create a heading in Markdown?

**Your answer:**


**Correct answer:** Use `#` symbols: `# Heading 1`, `## Heading 2`, `### Heading 3`, etc. More `#` symbols create smaller headings.
