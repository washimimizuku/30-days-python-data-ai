## Theory

### What is Jupyter?
Interactive computing environment:
- Mix code, output, and documentation
- Run code in cells
- See results immediately
- Great for exploration and teaching

### Cell Types
**Code Cells**: Execute Python code
**Markdown Cells**: Formatted text, headings, lists

### Keyboard Shortcuts
```
Shift + Enter  - Run cell and move to next
Ctrl + Enter   - Run cell, stay in place
Esc            - Command mode
Enter          - Edit mode
A              - Insert cell above
B              - Insert cell below
DD             - Delete cell
M              - Convert to markdown
Y              - Convert to code
```

### Magic Commands
```python
%matplotlib inline  # Display plots in notebook
%time code()       # Time execution
%load file.py      # Load code from file
%%timeit           # Time cell execution
```

### Best Practices
1. One concept per cell
2. Add markdown explanations
3. Clear outputs before committing
4. Use meaningful cell order
5. Document assumptions
