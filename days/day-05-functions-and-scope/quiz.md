# Day 5 Quiz: Functions

## Questions

### 1. What's the difference between parameters and arguments?

**Your answer:**

**Correct answer:** Parameters are variables in the function definition. Arguments are the actual values passed when calling the function.

---

### 2. What does this function return?
```python
def mystery(x):
    x = x + 1
    return
```

**Your answer:**

**Correct answer:** `None` - The function has a return statement but doesn't return any value.

---

### 3. What's a default argument?

**Your answer:**

**Correct answer:** A parameter with a default value that's used if no argument is provided. Example: `def greet(name="World"):`

---

### 4. Can a function return multiple values?

**Your answer:**

**Correct answer:** Yes, by returning a tuple: `return value1, value2, value3`. You can unpack them: `a, b, c = my_function()`

---

### 5. What's the scope of a variable defined inside a function?

**Your answer:**

**Correct answer:** Local scope - it only exists inside the function and can't be accessed outside.
