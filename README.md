## What You'll Learn

### Core Python (Days 1-10)
- Variables, data types, operators
- Control flow (if/else, loops)
- Functions and modules
- Lists, tuples, dictionaries, sets
- File I/O
- Error handling
- Object-oriented programming basics

### Data-Focused Python (Days 11-20)
- NumPy for numerical computing
- Pandas for data manipulation
- Data cleaning and transformation
- Working with CSV and JSON
- DateTime handling
- Regular expressions
- List comprehensions and generators

### AI/ML Python (Days 21-30)
- Virtual environments and package management (pip, uv, poetry)
- Jupyter notebooks for interactive computing
- Matplotlib for data visualization
- Working with APIs (requests library)
- Async programming with asyncio
- Type hints and Pydantic for data validation
- Testing with pytest
- Building complete ETL pipelines
