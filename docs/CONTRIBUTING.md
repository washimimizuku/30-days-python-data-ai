# Contributing to 30 Days of Python for Data and AI

Thank you for your interest in contributing! This project aims to provide high-quality Python education for data engineering and AI.

## How to Contribute

### Reporting Issues

Found a bug or typo? Have a suggestion?

1. Check if the issue already exists
2. Create a new issue with:
   - Clear title
   - Detailed description
   - Steps to reproduce (if applicable)
   - Expected vs actual behavior

### Suggesting Improvements

Have ideas for better explanations or examples?

1. Open an issue with the "enhancement" label
2. Describe your suggestion
3. Explain why it would improve the curriculum

### Submitting Changes

Want to fix a bug or improve content?

1. **Fork the repository**
2. **Create a branch**: `git checkout -b fix/your-fix-name`
3. **Make your changes**
4. **Test your changes**: Ensure code runs correctly
5. **Commit**: `git commit -m "Fix: description of fix"`
6. **Push**: `git push origin fix/your-fix-name`
7. **Create a Pull Request**

## Content Guidelines

### Code Quality

- **Working code**: All solutions must run without errors
- **Comments**: Explain complex logic
- **Style**: Follow PEP 8 guidelines
- **Testing**: Test code before submitting

### Documentation

- **Clear explanations**: Use simple language
- **Code examples**: Include practical examples
- **Consistency**: Match existing format and style
- **Accuracy**: Verify technical accuracy

### Exercises

- **Progressive difficulty**: Start simple, build complexity
- **Real-world relevance**: Use data engineering scenarios
- **Clear instructions**: Provide TODO comments
- **Complete solutions**: Include working solutions

## Review Process

1. **Submission**: Create pull request
2. **Review**: Maintainers review changes
3. **Feedback**: Address any comments
4. **Approval**: Changes are merged
5. **Credit**: You're added to contributors!

## Code of Conduct

### Our Standards

- **Be respectful**: Treat everyone with respect
- **Be constructive**: Provide helpful feedback
- **Be inclusive**: Welcome all skill levels
- **Be patient**: Remember everyone is learning

### Unacceptable Behavior

- Harassment or discrimination
- Trolling or insulting comments
- Personal attacks
- Spam or off-topic content
